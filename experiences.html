Apple Developer Academy

february/2015 - december/2015

O Apple Developer Academy (antes chamado BEPiD) é um programa com o objetivo de capacitar estudantes para a criação de aplicativos para iOS, desde o amadurecimento da ideia, passando pelo desenvolvimento em si e culminando na publicação do app na App Store.

globo.com

june/2016 - november/2017

Comecei a estagiar na globo.com em 2016, na área de reprodução de vídeos na plataforma mobile. Trabalhei principalmente com Swift (iOS), Java e Kotlin (Android). Após, trabalhei com backend web em Python (especialmente framework tornado), na plataforma de publicação de conteúdo dos produtos da globo.com. Nesta época também tive experiência com Javascript no frontend.
