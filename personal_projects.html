Camouflage Image

This project was developed as the final assignment of Fundamentalms of Image Processing class.

The goal here was to implement a basic versão of the image camouflage method described on the paper Camouflage Images presented at SIGGRAPH 2010. The source code is on github.

Crazy Helicopter Pilot Simulator

Swift ∙ Python

This project was developed as the final assignment of Human–Computer Interaction class.

It's an adaptation of a 3D helicopter simulator (forked from an open-source project) where the user can control it through hand movements, captured with the gyroscope of an iPhone.

The source code of the simulator adaptation and the mobile app is on github.


UFRGS History Plotter

Javascript ∙ R

This project allows UFRGS students to have a different visualization of their academic journey, by plotting graphs from their course history, obtained in the university's website. The source code is on github.

Memes top d+

Javascript ∙ Python

This is a pseudo-motivational meme bot. Using a pre-determined list of image tags and motivational quotes websites, the bot searches for an image and a quote and creates a random meme with them. Then, it is posted automatically on the facebook page.
